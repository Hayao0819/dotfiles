## Dotfiles

個人用Dotfilesです。

### 導入

[hayao0819/lico >= v0.4.0](https://github.com/Hayao0819/lico)を使う。


```bash
sudo sh -c "$(curl -L https://raw.githubusercontent.com/Hayao0819/lico/master/dl.sh)"
lico init "https://github.com/Hayao0819/dotfiles.git"
lico set
```
